Admission Module Instructions

  • This page is strictly for use by authorized admin and admissions staff.
  • Please ensure that all student details are entered accurately; all required fields must be completed.
  • The Subjects field allows multiple selections. Use Ctrl (or Cmd on Mac) to select more than one subject.
  • The Stream field offers: Arts, Commerce, Science, and Nothing. Choose the option that best describes the student's academic focus.
  • The SR Number is used to generate the unique student document ID (e.g., SR 5 becomes student00005).
  • An automatic timestamp is recorded upon submission.
  • Please confirm that the ₹500 admission fee has been received by checking the Admission Fee Checkbox before submitting.
  • All data management and updates in this module are the responsibility of the designated administrative staff.
