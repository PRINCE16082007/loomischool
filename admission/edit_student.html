Instructions for Editing Student Data

  • Use the Search box to find students by name, SR Number, mobile, or Aadhar.
  • Apply Filters (Class, Stream, Category) to narrow down the list.
  • Click on a student from the list to load their data for editing.
  • All fields marked with * are required.
  • SR Number and Admission Fee Status are read-only and cannot be changed.
  • For Class 9/10, Stream is set to N/A and only Sanskrit/Punjabi subjects are available.
  • For Class 11/12, select Stream first, then choose from stream-specific subjects.
  • Click "Update Student Data" to save changes to the database.
